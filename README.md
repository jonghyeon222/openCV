## 첫 번째 실습과제: 그림판 만들기
OpenCV의 마우스 이벤트와 키보드 입력을 활용하여 간단한 디지털 캔버스를 구현한 실습입니다.

1. 주요 기능
자유 드로잉: 마우스 왼쪽 버튼을 누른 상태로 움직이면 검은색 선이 그려집니다.
캔버스 초기화: 키보드 c 키를 누르면 화면이 다시 하얗게 지워집니다.
프로그램 종료: 키보드 q 또는 ESC 키를 누르면 창이 닫히고 프로그램이 종료됩니다.

2. 기술 스택 및 환경
Language: Python 3.9
Library:
 - OpenCV (4.13.0): 창 관리 및 마우스/키보드 이벤트 처리
 - Numpy (2.0.2): 512x512 크기의 하얀색 캔버스(행렬) 생성

3. 핵심 코드 설명
마우스 이벤트 처리:
 - cv2.EVENT_MOUSEMOVE와 cv2.EVENT_FLAG_LBUTTON의 비트 연산(&)을 통해 드래그 상태를 확인합니다.
키보드 이벤트 처리:
 - cv2.waitKey(0)를 통해 입력을 대기하며, ord('q')와 27(ESC)을 비교하여 종료 로직을 수행합니다.
 - if (key == ord('q') or key == 27): 구문을 사용하여 다중 종료 키를 지원합니다.

4. 실행 방법 
가상환경(Python 3.9) 활성화
pip install opencv-python numpy 설치
파이썬 파일 실행 후 마우스로 자유롭게 그림 그리기


## 두 번째 실습과제: 비트 연산을 활용한 이미지 합성

OpenCV의 HSV 색상 공간 변환과 비트 연산(AND, OR, NOT)을 활용하여 특정 배경을 제거하고 두 이미지를 합성하는 크로마키 기법을 구현한 실습입니다.

1. 주요 기능
색상 기반 마스킹: 특정 색상 범위(초록색)를 지정하여 배경을 분리하는 마스크를 생성합니다.
객체 추출 및 배경 제거: 생성된 마스크를 반전시켜 캐릭터 본체만 추출하고, 배경 이미지에서 해당 영역을 비워둡니다.
최종 이미지 합성: 추출된 캐릭터 본체와 영역이 확보된 배경 이미지를 결합하여 하나의 결과물을 만듭니다.

2. 기술 스택 및 환경
Language: Python 3.9
Library:
 - OpenCV (4.13.0): 이미지 프로세싱 및 비트 연산 처리
 - Numpy (2.0.2): 색상 범위 설정을 위한 배열(Array) 생성 및 행렬 연산

3. 핵심 코드 설명
색상 공간 변환 및 마스크 생성:
 - cv2.cvtColor(sonic, cv2.COLOR_BGR2HSV)를 통해 조명 변화에 강한 색상 검출 환경을 만듭니다.
 - cv2.inRange() 함수로 초록색 범위를 지정하여 배경은 흰색(255), 객체는 검은색(0)인 이진 마스크를 생성합니다.
비트 연산을 통한 합성 로직:
 - cv2.bitwise_not(mask1)을 통해 캐릭터 영역을 선택하는 반전 마스크를 생성합니다.
 - cv2.bitwise_and()로 배경이 제거된 캐릭터와 캐릭터 자리가 비워진 배경 이미지를 각각 준비합니다.
 - cv2.bitwise_or()를 사용하여 두 이미지를 병합함으로써 최종 합성 이미지를 도출합니다.

가상환경(Python 3.9) 활성화
